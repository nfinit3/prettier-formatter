# prettier-formatter

prettier-formatter is my personal prettier formatting configuration to be used with text editors.

## Installation

**Install using Yarn:**

```bash
yarn add --dev prettier-formatter
```

**Install using npm:**

```bash
npm i prettier-formatter --save-dev
```

## Usage

Edit ::package.json::

```jsonrc
"prettier": "prettier-formatter"
```
